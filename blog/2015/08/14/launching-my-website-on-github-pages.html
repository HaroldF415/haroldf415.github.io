Website launched using Github Pages.

Posted on: August 14, 2015

Finally used the improved Github Windows application to launch my personal website. I feel like I'm cheating due to the fact that I was not using terminal to accomplish this task and rather relied on the fancy UI GitHub provide. Upon using github pages I noticed that PHP files are not supported. Hence rendering my contact form useless. Page was taken out immediately. Next up I will link this personal website to Design Critiques from reddit to receive some constructive criticism. Hopefully I will receive some good ideas and useful advice that will help me better myself and this website. There are still alot of work to be done on this website. Truthfully I think I lack semantically correct HTML. Even more so the css could be condensed even more.
