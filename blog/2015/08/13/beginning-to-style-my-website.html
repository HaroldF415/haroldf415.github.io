Styling "About Me" section.

Posted on: August 13, 2015

Making some progress. Managed to finally get the style that I wanted for my blog. Things will be added later to make it more user friendly but for now it will stay like this. Was thinking that all my pages have a different CSS file and maybe I could condense it into one file. But at the moment it does not seem practical. Since I have media queries doing different things on each page when the screen is resized. Friend of mine mentioned that the colors used for my site were a bit off. Also thinking about changing the way my "ABOUT ME" section looks like. I feel it is kind of a gimmick rather than showing my true skills. Had a problem with the contact_index.php where the form was not resizing when the screen became smaller. So I inserted input, textarea{ max-width: 380px; } into contact.css. Found a need trick for .conIcons on the index.html page [Home Page]. When clicking them they will now push in ever so slightly. Running into a snag with gishwhes.html. NAV in that page does not scroll.
