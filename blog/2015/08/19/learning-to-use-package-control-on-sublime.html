Package Control, ColorSublime & Sublime Text 3

Posted on: August 19, 2015

While working with Sublime text I've realized that the default color settings were putting a strain on my eyes, so I looked to change the color scheme. Here is how I tried accomplishing that and almost gave up becuase I kept running into trouble everys step of the way.

  • Step 1: Installing Package Control.

    This was a step out of my comfort zone because until now I've always installed things by putting them inside folders and restarting the app. Package Control on the other hand requires the use of Sublime Text console (I tend to stay away from many console driven procedures). Following the instructions set here

  • Step 2: Installing Color Sublime

    Searching for a decent pluging to change the scheme of my text editor I decided on Color Sublime. Following their instructions here: Installing Color Sublime it was clear that something went wrong even though in their site they promote:

    "Installing a ColorSublime theme on Sublime Text 2/3 is really simple."

    Nontheless I might have glossed over the part where it says "theme". Anyways package control now only supports ColorSublime in Sublime Text 3 look. So now I have to install Sublime Text 3

  • Step 3: Installing Sublime Text 3

    Installing Sublime Text 3 wasn't so much of a hassle, but making it my defaul editor was. It seems that when uninstalling Sublime Text 2 (my current text editor) the Sublime Text Uninstaller doesn't remove a particular registry entry referring to Sublime Text 2's sublime_text.exe, which prevents Windows from adding an entry for Sublime Text 3's sublime_text.exe. One way to fix this is manually, open regedit > HKEY_CLASSES_ROOT\Applications > sublime_text.exe entry, and delete it. After following these steps I managed to make Sublime 3 my default text editor.

  • Step 4: Repeat Steps 1 & 2

    Now that Sublime Text 3 has been installed and has been made into my default text editor following Steps 1 & 2 was much easier and picking a color scheme that doesn't put a lot of strain on my eyes was a breeze. Decided to go with "BatmanLight" why you ask? Simple, because...
