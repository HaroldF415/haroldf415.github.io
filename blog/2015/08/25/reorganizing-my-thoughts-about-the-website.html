One awesome tip that I've learned is:

  • Document Object Model: (DOM)

    DOM is a description of how eveything on a webpage that we are looking at works together.

  • Block Level Elements:

    Create larger structure than inline elements. Block level elements may contain inline elements and other block level elements. Separate content and creates blocks in a layout.

  • Inline Level Elements:

    Only contain data and other inline elements. By default they do not create a new line when rendered in the browser.

  • Inline-Block:

    Sits inline with the content, but they can have the characteristics of a block. Meaning they can have propertions: width, height, padding and margin.
