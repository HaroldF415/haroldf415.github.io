Writing code on the road.

Posted on: August 12th, 2019

This idea did not pan out well. Came to the realization that while I'm working Uber at nights the only thing that I can do is listen to YouTube videos about programming. Driving and taking breaks to program did not work well. I would start a thought on how to write some code and then I'd get a ping to go pick up a rider. Another method of studying must be implemented.
